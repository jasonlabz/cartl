等价规则

必须保留

  • 30 个可访问路由及详情/版本 URL 参数
  • 原 API 路径、请求字段、分页、令牌刷新和业务错误语义
  • 搜索、筛选、分页、编辑、发布、导入导出、画布编辑等可见交互
  • 原 Vue 的后台信息密度、层级、响应式和状态反馈

不会带入

  • Vue、Pinia、Naive UI 与原组件实现
  • 无业务价值的框架兼容层
  • 对后端契约、数据库和原 Vue 源码的修改
  • 仅为未来猜测预留的抽象

实现时使用 Ant Design 作为控件基线，保留现有 React Flow、Recharts、Lucide；视觉以原 Vue 页面为准而不是沿用当前原型的自定义数据。
